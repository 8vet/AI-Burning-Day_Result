## 멍탐정 

<hr>

+ 팀명: 완두콩 넷
+ 팀원: 이동은, 오은택, 박효진, 김진우

<br>

### 그 강아지는 몇 살일까요?

<hr>

+ 사진 한장이면 그 강아지의 나이를 알려드립니다!
+ 견종에 따른 맞춤 모델을 사용한 멍탐정!
+ 사진 속 강아지의 견종, 나이에 맞는 정보도 제공해드립니다!

<br>

### 사용 방법

<hr>

+ 사진 업로드 (끝)
+ 정보 받기 (끝)
+ Easy!
