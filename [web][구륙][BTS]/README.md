# __BTS - Best Translation Service__

papago NMT API를 활용한 고품질 번역 유튜브 학습 보조 서비스  

___
## Team  
구륙 - 김태형, 장민호, 김동진, 채지훈

___
## Description

유튜브 자동생성 영어 자막을 Sentence Boundary Detection(SBD), Papago NMT를 활용하여 한글 자막을 보여주는 웹 서비스.
